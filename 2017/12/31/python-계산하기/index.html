계산하기
  • 파이썬은 실행후 결과를 변수에 담지 않으면 바로 사라집니다. repl에 출력되는 결과는 메모리에 출력된 결과를 보여 주는겁니다. 화면에 출력되는 것이 아닙니다.
  • 모니터 화면에 출려하기 위해서는 print() 함수를 사용합니다.
  • 파이썬3에서 /는 기본적으로 실수로 표기 됩니다. 파이썬2에서는 정수표기 됩니다.

자료형

  • 정수형

    • 정수형은 기본적으로 28byte 크기를 갖습니다. 값이 크면 자동으로 크기를 키웁니다. 최대값의 크기를 생각하지 않고 사용해도 됩니다. 최대 400byte까지 키울 수 있습니다. 64bit, 32bit 운영체제에 따라 기본 크기가 다릅니다.

  • 실수형

    • 실수형도 정수형처럼 필요하다면 자동으로 크기를 키우기 때문에 자유롭게 사용하면 됩니다. 참고로 기본 24byte(최대 400byte)의 크기를 갖습니다. 다른 프로그래밍 언어와 실수를 처리하는 방법이 다르기 때문에 일반 수학의 상식에 따라 사용하면 됩니다. c언어 에서 (1/3) + (1/3) + (1/3) = 0.9999999999 가 되지만 파이썬은 1.0이 됩니다. 또한 0.3 + 0.3 + 0.3 + 0.1 = 0.9999999999999 가 됩니다. 항상 round(실수, 반올림자리)와 사용해야 합니다.

  • 문자열형

    • ', ", ''', """ 로 감싸면 문자열이 됩니다.
    • 영문으로만 구성될 때는 49byte + 영문수, 한글이 포함 될때는 74 + (문자수*2)
    • 인덱싱
      문자열은 배열처럼 인덱싱이 가능하다.

    • 슬라이싱
      문자열은 슬라이싱이 가능하다.
