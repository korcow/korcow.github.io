주피터 사용법

위와 같이 주피터를 처음 실행시키면 주피터 탐색기가 실행 됩니다. 원하는 디렉토리로 이동하거나 새로운 폴더를 만들 수 있습니다.

실행 중인 노트북이나 클러스터를 확인 할 수 있습니다.

새파일 만들기


위 처럼 우측에 새파일 만들기 => python3를 선택하면 새로운 python3파일이 생성됩니다.

이름바꾸기


위 그림처럼 Untitled를 클릭하면 이름을 바꿀수 있습니다.

전체적으로 직관적이고 사용법이 사용하기 쉽습니다.
<F1> 을 누르면 도움말이 나옵니다.

메뉴와 툴바는 직접 확인해 보세요.

mode

주피터 셀은 3가지 모드가 있습니다.

  • code
    파이썬 코드를 입력할 수 있습니다.

  • 툴바의 키보드 모양 왼쪽에 보면 code로 되어 있습니다.
    클릭해서 markdown과 raw로 바꿀수 있습니다.

  • raw는 ascii파일 입니다.

  • markdown
    마크다운 문서를 만들 수 있습니다.

기본 사용법

먼저 code mode로 python code를 입력해보세요.

shift + Enter 하면 해당 셀이 실행되고 한줄이 추가 되어 집니다.
별도로 다시 실행 하려면 윈도우는 Ctrl+Enter 맥은 커맨드+엔터 하면 실행 됩니다.

이번에는 두번째 셀을 markdown mode로 바꾸고 마크다운을 입력해 보세요.

코드블럭도 사용할 수 있습니다.

raw 나 head line은 직접 해보세요.
