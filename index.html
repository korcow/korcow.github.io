Hexo 한글파일명 문제점

2017-11-03

Hexo에서 _Post의 관리는 hexo generater가 md파일을 html로 만들어주고 .deploy디렉리에 add, commit 을 하고, ddeploy는 push 동작을 한다. 추측이지만 generater가 한글파일의 감시를 제대로 못해주는 것 같다. 감시는 파일명감시, 내용수정감시로 나뉘어 지는 것같다.

      shell이란

      2017-10-31

      리눅스 시스템이 부팅이 완료되고 로그인을 하면 커서가 깜빡이며 명령 수행을 대기합니다. 이것을 shell prompt 라고합니다.
      shell은 interpreter 입니다. 명령을 해석하고 올바른 명령이라면 kernel로 명령을 전달하고 처리 합니다.

      shell이 interpreter라고 했습니다.
